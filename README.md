# m321-chat

A real-time Chat application to learn Microservices

![app](data/chat.png)

## Getting Started

### Prerequisites

- Java 21 (GraalVM)
- Gradle
- Docker

### Installation

Build a microservice:

```bash
gradle build
```

Run a microservice:

```bash
gradle quarkusDev
```
